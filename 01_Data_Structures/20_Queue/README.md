### 📌 Queue (큐)

- **시간 복잡도 / Time Complexity**: 삽입(Enqueue) & 삭제(Dequeue): $O(1)$
- **공간 복잡도 / Space Complexity**: $O(N)$
- **핵심 개념 / Core Concept**: FIFO(First-In, First-Out) 구조로, 가장 먼저 삽입된 데이터가 가장 먼저 삭제되는 선형 자료구조입니다. / A linear data structure operating on the FIFO principle, where the first element added is the first to be removed.
- **적용 상황 / When to Use**: BFS(너비 우선 탐색)의 순서 보장, 프로세스 스케줄링(선입선출 작업), 대기열 관리(프린터 스풀, 티켓 예매 시스템), 버퍼(Buffer) 및 메시지 큐에 사용합니다. / Used for maintaining order in BFS (Breadth-First Search), process scheduling (FIFO task processing), queue management (printer spools, ticketing systems), and data buffers.
