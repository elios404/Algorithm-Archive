### 📌 Stack (스택)

- **시간 복잡도 / Time Complexity**: 삽입(Push) & 삭제(Pop): $O(1)$
- **공간 복잡도 / Space Complexity**: $O(N)$
- **핵심 개념 / Core Concept**: LIFO(Last-In, First-Out) 구조로, 가장 나중에 삽입된 데이터가 가장 먼저 삭제되는 선형 자료구조입니다. / A linear data structure operating on the LIFO principle, where the last element added is the first to be removed.
- **적용 상황 / When to Use**: 역순 정렬, 작업 취소(Ctrl+Z), 웹 브라우저 뒤로가기, 수식 괄호 검사, 재귀 호출 및 DFS 역추적(Backtracking) 시 사용합니다. / Used for reversing sequences, undo operations, browser navigation history, bracket matching, and backtracking in recursion/DFS.
